        # 产品介绍

        OC-工时系统针对公司内部OC部门全体人员，全体人员都必须填报工时，工时汇总之后报送给财务部门。OC工时分为两大类：项目工时和通用项目工时，其中，项目工时是涉及到合同的项目所花费的工时，项目工时由项目经理进行审核；通用项目工时是与合同无关的日常工作工时。

        # 我是项目经理

        # 项目经理做哪些事情？

        （1）项目经理可以给工程师分配项目任务，工程师填写任务工时之后，项目经理负责审核工程师填报的工时。

        （2）项目经理填报本人的工时，当项目经理工作涉及合同类项目任务，需要填写项目任务工时，该项目工时由本人进行审核处理；当项目经理工作涉及非合同类项目（也就是日常工作），需要填写通用项目任务工时，由指定审核人进行通用项目工时审核。

        # 项目经理给工程师分配项目任务

        （1）选择需要下发任务的项目

        项目经理打开“项目列表”菜单，根据项目分类（通用以外的项目）和项目筛选功能，快速查找需要下发任务的项目。如图2-2

        图 2-2

        查找到相应项目后，点击项目名称，进入到“计划任务”页面（也可以直接点击“计划任务”菜单，在页面上方选择项目）。如图2—3,2-4

        图 2-4

        （2）添加项目任务

        先在左侧任务树选择一个任务节点，右侧列表中点击【添加】，则任务添加到该节点下。如图2-5

        图 2-5

        （3）填写任务信息并添加任务成员

        点击【添加】，选择所需的人员，安排工时与具体工作内容。如图2-6

        图 2-6

        （4）发布项目任务

        任务添加后，勾选项目任务，然后点击【发布】，相应人员即可接收该任务。如果项目没有进行发布，任务接收人看不到该项目任务。如图2-7

        # 我是工程师

        # 工程师做哪些事情

        （1）工程师做合同类项目任务时，按照项目经理分配的项目任务填写每天的工时。并由项目经理对其工时进行审核。

        （2）工程师做日常工作（非合同项目工作），由指定审核人进行审核其日常工时。

        # 填报本人工时

        # 我非项目人员

        # 非项目人员做哪些事情？

        非项目人员是指日常不做合同类项目工作，比如客服、前台、智能运维部门、售后部门、资产管理部等部门人员。

        （1）非项目人员需填报本人通用项目工时，选择通用项目进行填写，其工时由指定审核人进行审核。

        （2）当非项目人员参与合同项目时，那么由项目经理发合同项目任务，非项目人员填写工时，该项目工时由项目经理进行审核。

        （3）普通非项目人员不能发任务给其他人。

        # 非项目人员做哪些事情？

        （1）填写通用项目工时

        非项目人员可以在计划外任务列表填报工时。在计划外任务列表点击【添加】，生成相应的任务，进行工时填报。如图2-23

        图 2-24

        图 2-25

        # 我是区域总监、部门负责人或主管

        # 区域总监、部门经理或者主管做哪些事情？

        （1）区域总监、部门经理和主管需要审核下属的日常工作（通用项目）工时

        （2）区域总监、部门经理和主管需要填写本人日常工作（通用项目）工时，原则上由其上级对工时进行审核。

        （3）区域总监、部门经理和主管如果涉及到合同类项目任务，由项目经理发任务，部门经理和主管填写项目工时，项目经理对该任务项目工时进行审核。

        # 审核他人工时

        区域总监、部门经理和主管点击“报工审核”菜单，选中通用类项目的填报记录（在列表中标记为“计划外”），点击【通过】或【驳回】按钮，进行审核。审核支持批量操作。如图2-28
